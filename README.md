# 회원가입 데모

HTML/CSS와 JavaScript를 이용한 간단한 회원가입 창 만들기 프로젝트

![image](https://user-images.githubusercontent.com/30149272/105858258-73a6b780-602e-11eb-910f-b0d09dae35a2.png)

<주요 기능>
- 입력 여부 검사
- 정규표현식을 통한 유효성 검사
- 비밀번호와 비밀번호 확인 일치 여부 검사

<개선 방향>
- 데이터베이스에 회원 정보 저장
- 저장된 회원 정보를 통해 등록 계정 유효성 검사
